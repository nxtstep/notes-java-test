# Notes
*io.supersimple.notes*  

Android App to demonstrate Automated testing on Android with Dagger2, Mockito and RxJava2.  

### Build

#### Debug
`./gradlew assembleDebug`

#### Release
`./release.sh`

#### Checkstyle  
`./gradlew checkstyle`

#### Outdated dependecies
`./gradlew dependencyUpdates -Drevision=release`
